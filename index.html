Collaboration with MistBot: The Beginning

Welcome to the official integration guide for MistBot. This project introduces a multi-server trade link system designed to synchronize the marketplace experience across the Discord ecosystem.

Key Highlights

  • Global Trade Link: Post offers that appear across all participating servers instantly.
  • Discovery Engine: Advanced search functionality to find specific buyers and sellers.

Marketplace Interaction

Users can interact with the global trading hub using the following commands:

Command Description
,market Entry point for all marketplace interactions.
post Broadcasts a trade offer across all servers in the feed.
search trades Search for specific items or "Looking For" requests globally.

Bump Mechanic

To maintain a fresh marketplace, un-bumped offers are removed after 6 hours. Bumping an offer resets this timer and moves it to the top of the global trading feed.

Staff Administration

Individual Configuration

  • /set-feed: Establishes the live trade feed for the server.
  • /settradechannel: Restricts market commands to a specific channel.
